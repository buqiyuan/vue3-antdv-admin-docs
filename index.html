高效快速

vite5.x 加持，开发、打包、热更新、热加载、热替换，一切都是那么的快

预设样式

更快、更上层的原子化 CSS 引擎，听说，下雨天，属性化书写 class 名和 vue3 更配哦

TypeScript

项目全量使用 TypeScript，代码处处有提示，老大再也不用担心我叕写错变量名了

MSW Mock 服务

项目使用 MSW 作为 Mock 服务，在浏览器端也能像后端一样编写 API 服务

状态管理

项目使用 Pinia 作为状态管理，使用起来更加简单

全栈式
